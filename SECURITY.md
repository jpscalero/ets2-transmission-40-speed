<a name="english"></a>
## 🇺🇸 English: Supported Versions
We only support the latest stable version of Euro Truck Simulator 2 (v1.58.x).

## Reporting a Vulnerability
If you find a security issue, please open a private [Issue](../../issues) or contact the maintainers. Since this is a game mod, security risks are minimal, but we take them seriously.

<a name="espanol"></a>
## 🇪🇸 Español: Versiones Soportadas
Solo damos soporte a la última versión estable de Euro Truck Simulator 2 (v1.58.x).

## Reportar una Vulnerabilidad
Si encuentras un problema de seguridad, por favor abre un [Issue](../../issues) privado o contacta a los mantenedores. Al ser un mod de juego, los riesgos son mínimos, pero los tomamos en serio.
