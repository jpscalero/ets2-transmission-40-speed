# Historial de Cambios (Changelog) - Antigravity 40-Speed

Todos los cambios notables en este proyecto serán documentados en este archivo. El formato se basa en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.5.8] - 2026-04-30
### Añadido
- **Automatización CI/CD**: Implementación de GitHub Actions para empaquetado automático de archivos `.scs`.
- **Ecosistema Profesional**: Nuevas plantillas de Issues, Código de Conducta y Política de Seguridad.
- **Wiki Técnica**: Documentación profunda sobre la filosofía de ingeniería de 40 marchas.
- **Banners e Iconos**: Arte profesional generado para el repositorio y el mod.

### Cambiado
- **Localización**: El repositorio ahora es 100% en español.
- **Identidad**: Juan Pedro ha sido designado como Auditor y Mantenedor Principal.
- **Estructura del Mod**: Optimización de la jerarquía de carpetas para compatibilidad total con ETS2 v1.58.

### Corregido
- **Depreciación de GHA**: Se corrigieron errores de versiones obsoletas en los workflows de GitHub.
- **Invisibilidad en Taller**: Solucionado el problema donde las transmisiones no aparecían debido a IDs de accesorios excesivamente largos.
